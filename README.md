# Validation

Cours Validation @ ENSTA Bretagne.

## Installation

- Python 3.10

1. Clone the repo :

```bash
git clone git@github.com:CastelledSnake/Validation
```

2. Run the `graphs.py` file

```bash
python3 graphs.py
```
